## 1. 분석 개요

### 1.1 목적
- 브랜드별/채널별 매출채권 현황 파악
- 회전일수(DSO) 분석을 통한 채권 회수 효율성 평가
- 운전자본 관점에서 매출채권이 현금흐름에 미치는 영향 분석

### 1.2 핵심 지표

| 지표 | 산식 | 의미 |
|:---|:---|:---|
| **DSO (Days Sales Outstanding)** | 매출채권 ÷ 일평균매출 | 채권 회수에 걸리는 평균 일수 |
| **매출채권회전율** | 연매출 ÷ 매출채권 | 연간 채권 회수 횟수 |
| **채권증가율** | (당기채권-전기채권) ÷ 전기채권 | 채권 증감 비율 |

### 1.3 매출 기준 정의

```
┌─────────────────────────────────────────────────────────────────────┐
│  ⭐ 매출채권 회전일수(DSO) 계산 시 매출 기준                        │
│                                                                     │
│  【실판매 (V-)】 = VAT_EXC_ACT_SALE_AMT                             │
│  - COPA VAT 제외 실판매액                                           │
│  - 재무제표 대시보드 작성가이드와 동일한 기준                       │
│                                                                     │
│  ※ 재무제표 대시보드 회전율 분석 기준:                             │
│  - DSO = 매출채권 ÷ (실판매(V-) ÷ 365)                             │
│  - DIO = 재고 ÷ (출고매출(V-) ÷ 365)                               │
│  - DPO = 매입채무 ÷ (매출원가 ÷ 365)                               │
└─────────────────────────────────────────────────────────────────────┘
```

    SELECT 
        CALMONTH,
        CASE WHEN WWDCH = '09' THEN 'EXPORT' ELSE 'DOMESTIC' END AS CHANNEL,
        SUM(TRY_TO_NUMBER(COL_2_TOTAL)) AS AR_AMT

      AND ZARTYP = 'R1'
      AND WWBND IN ('M', 'I', 'X', 'V', 'ST', 'W')
    GROUP BY CALMONTH, CASE WHEN WWDCH = '09' THEN 'EXPORT' ELSE 'DOMESTIC' END
),
sale_data AS (
    -- 실판매(V-) = VAT_EXC_ACT_SALE_AMT
    SELECT 
        CASE WHEN CHNL_CD = '9' THEN 'EXPORT' ELSE 'DOMESTIC' END AS CHANNEL,
        SUM(VAT_EXC_ACT_SALE_AMT) AS SALE_AMT
    FROM FNF.SAP_FNF.DW_COPA_D
    WHERE PST_DT >= '2025-01-01' AND PST_DT <= '2025-11-30'
      AND CORP_CD = '1000'
      AND BRD_CD IN ('M', 'I', 'X', 'V', 'ST', 'W')
      AND CHNL_CD NOT IN ('99', '0')
    GROUP BY CASE WHEN CHNL_CD = '9' THEN 'EXPORT' ELSE 'DOMESTIC' END
)
SELECT 
    a.CHANNEL,
    ROUND(a.AR_AMT / 100000000, 1) AS AR_100M,
    ROUND(s.SALE_AMT / 100000000, 1) AS SALE_100M,
    ROUND(a.AR_AMT * 334 / NULLIF(s.SALE_AMT, 0), 0) AS DSO_DAYS
FROM ar_data a
JOIN sale_data s ON a.CHANNEL = s.CHANNEL

## 7. DSO 계산 방법

### 7.1 산식

```
DSO = 매출채권잔액 × 분석기간일수 ÷ 실판매(V-)

【실판매(V-)】 = VAT_EXC_ACT_SALE_AMT (COPA VAT 제외 실판매액)

예시 (25년 11월, 국내):
DSO = 1,006억 × 334일 ÷ 7,050억 = 48일
```

### 7.2 분석기간 일수

| 기준월 | 일수 | 산출근거 |
|:---:|:---:|:---|
| 1월 | 31일 | 1월 말 기준 |
| 6월 | 181일 | 1~6월 합계 |
| 11월 | 334일 | 1~11월 합계 |
| 12월 | 365일 | 연간 |

### 7.3 연환산 DSO

```
연환산 DSO = 매출채권 × 365 ÷ 연환산 실판매(V-)

연환산 실판매(V-) = 당년 1~11월 실판매 + 전년 12월 실판매

※ 재무제표 대시보드 회전율 분석과 동일한 기준
```
