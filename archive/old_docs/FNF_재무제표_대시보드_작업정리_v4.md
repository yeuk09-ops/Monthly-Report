## 1. 대시보드 개요

### 1.1 목적
F&F(FNF) 재무 성과를 월별로 분석하고 경영진에게 핵심 지표를 제공하는 관리회계 기반 대시보드

### 1.2 데이터 기준
- **기간:** 24년 1월 ~ 25년 11월 (월별 누적)
- **기준일:** 매월 말일 기준
- **연환산:** 당해년도 1~11월 + 전년도 12월

### 1.3 관리손익 vs 재무손익 개념

```
┌─────────────────────────────────────────────────────────────────────┐
│  ⚠️ 중요: 관리손익과 재무손익의 차이                                 │
│                                                                     │
│  【관리손익 (CO 관점)】- 본 대시보드 기준                           │
│  ├── 매출액 (실판매 V-)                                             │
│  ├── (−) 점수수료           ← 매출에서 차감                         │
│  ├── = 출고매출 (V-)        ← 관리목적상 순매출 개념                 │
│  ├── (−) 매출원가                                                   │
│  ├── = 매출총이익                                                   │
│  ├── (−) 판관비 (점수수료 제외)                                     │
│  └── = 영업이익                                                     │
│                                                                     │
│  【재무손익 (외부보고 관점)】                                       │
│  ├── 매출액 = 실판매 (출고매출 개념 없음)                           │
│  ├── (−) 매출원가                                                   │
│  ├── = 매출총이익                                                   │
│  ├── (−) 판관비 (점수수료 포함)   ← 판관비로 이동                   │
│  └── = 영업이익                                                     │
│                                                                     │
│  ✅ 영업이익은 동일, 표시 방식만 다름                               │
└─────────────────────────────────────────────────────────────────────┘
```

## 4. 재무상태표 탭 상세

### 4.1 B/S 구조

```
자산총계
├── 유동자산
│   ├── 현금및현금성자산
│   ├── 매출채권
│   ├── 재고자산
│   └── 기타유동자산
└── 비유동자산

부채총계
├── 유동부채
│   ├── 매입채무
│   ├── 단기차입금
│   └── 기타유동부채
└── 비유동부채

자본총계
├── 자본금
└── 이익잉여금
```

### 4.2 운전자본 분석

```
순운전자본 (NWC) = 재고자산 + 매출채권 − 매입채무
```

## 6. 재무비율 산식 정의

### 6.1 안정성 지표

| 비율 | 산식 | 24년 | 25년 |
|------|------|-----:|-----:|
| **부채비율** | 부채 ÷ 자본 × 100 | 34.8% | 23.2% |
| **자기자본비율** | 자본 ÷ 자산 × 100 | 74.2% | 81.2% |
| **순차입금비율** | (차입금 − 현금) ÷ 자본 × 100 | 2.1% | △12.5% |

**계산 예시 (25년 11월):**
```
부채비율 = 4,163 ÷ 17,928 × 100 = 23.2%
자기자본비율 = 17,928 ÷ 22,091 × 100 = 81.2%
순차입금비율 = (0 − 2,245) ÷ 17,928 × 100 = △12.5%
```

### 6.2 수익성 지표 (연환산 기준)

| 비율 | 산식 | 24년 | 25년 |
|------|------|-----:|-----:|
| **영업이익률** | 영업이익 ÷ 실판매 × 100 | 26.1% | 30.1% |
| **매출총이익률** | (매출총이익 + 점수수료) ÷ 실판매 × 100 | 64.8% | 63.9% |
| **ROE** | 영업이익(연환산) ÷ 자본 × 100 | 27.2% | 28.0% |
| **ROA** | 영업이익(연환산) ÷ 자산 × 100 | 20.2% | 22.7% |

**계산 예시 (25년):**
```
영업이익률 = 5,013 ÷ 16,662 × 100 = 30.1%
매출총이익률 = (8,799 + 1,843) ÷ 16,662 × 100 = 63.9%
ROE = 5,013 ÷ 17,928 × 100 = 28.0%
ROA = 5,013 ÷ 22,091 × 100 = 22.7%
```

## 7. Snowflake 데이터 추출 로직

### 7.1 데이터베이스 정보

```
서버: FNF Snowflake
스키마: FNF.SAP_FNF (COPA 데이터)
       FNF.PRCS (운영 데이터)
```

### 7.2 손익 데이터 쿼리 (COPA 기반)

#### 7.2.1 매출액 및 비용 집계

```sql
-- 월별 손익 데이터 추출
SELECT 
    TO_CHAR(PST_DT, 'YYYY') AS YR,
    TO_CHAR(PST_DT, 'MM') AS MM,
    
    -- 매출액 (실판매 V-)
    ROUND(SUM(VAT_EXC_ACT_SALE_AMT) / 100000000, 0) AS SALE_AMT,
    
    -- 점수수료
    ROUND(SUM(SALE_CMS) / 100000000, 0) AS SALE_CMS,
    
    -- 출고매출 (실판매 - 점수수료)
    ROUND(SUM(VAT_EXC_ACT_SALE_AMT - SALE_CMS) / 100000000, 0) AS NET_SALE,
    
    -- 상품원가
    ROUND(SUM(ACT_COGS) / 100000000, 0) AS ACT_COGS

FROM FNF.SAP_FNF.DW_COPA_D
WHERE PST_DT >= '2024-01-01' AND PST_DT <= '2025-11-30'
GROUP BY TO_CHAR(PST_DT, 'YYYY'), TO_CHAR(PST_DT, 'MM')
ORDER BY YR, MM
```

#### 7.2.2 브랜드별 매출 (국내 vs 수출)

```sql
-- 브랜드별 국내/수출 매출 분리
SELECT 
    LEFT(TO_CHAR(PST_DT, 'YYYYMM'), 4) AS YR,
    BRD_CD,
    
    -- 전체 매출
    ROUND(SUM(VAT_EXC_ACT_SALE_AMT) / 100000000, 0) AS TOTAL_SALE,
    
    -- 수출 매출 (CHNL_CD = '9')
    ROUND(SUM(CASE WHEN CHNL_CD = '9' 
              THEN VAT_EXC_ACT_SALE_AMT ELSE 0 END) / 100000000, 0) AS EXPORT_SALE,
    
    -- 국내 매출 (수출 제외)
    ROUND(SUM(CASE WHEN CHNL_CD != '9' 
              THEN VAT_EXC_ACT_SALE_AMT ELSE 0 END) / 100000000, 0) AS DOMESTIC_SALE

FROM FNF.SAP_FNF.DW_COPA_D
WHERE PST_DT >= '2024-01-01' AND PST_DT <= '2025-11-30'
  AND TO_CHAR(PST_DT, 'MM') <= '11'
GROUP BY LEFT(TO_CHAR(PST_DT, 'YYYYMM'), 4), BRD_CD
ORDER BY YR, TOTAL_SALE DESC
```

#### 7.2.5 판관비 상세 쿼리

```sql
-- 판관비 항목별 집계
SELECT 
    TO_CHAR(PST_DT, 'YYYY') AS YR,
    
    -- 점수수료
    ROUND(SUM(SALE_CMS) / 100000000, 0) AS SALE_CMS,
    
    -- 기타수수료 (SM_CMS + CARD_CMS + ONLN_CNS_SALE_CMS)
    ROUND(SUM(SM_CMS + CARD_CMS + ONLN_CNS_SALE_CMS) / 100000000, 0) AS OTHER_CMS,
    
    -- 인건비
    ROUND(SUM(DMGMT_SALE_STFF_CNTRC_CST + DF_SALE_STFF_CNTRC_CST + 
              DS_SALE_STFF_CNTRC_CST) / 100000000, 0) AS LABOR,
    
    -- 물류비
    ROUND(SUM(LGST_CST) / 100000000, 0) AS LGST,
    
    -- 임차료
    ROUND(SUM(SHOP_FXD_RNT + SHOP_VRBL_RNT) / 100000000, 0) AS RENT,
    
    -- 감가상각비
    ROUND(SUM(DEPR) / 100000000, 0) AS DEPR,
    
    -- 관리비
    ROUND(SUM(MGMT_CST) / 100000000, 0) AS MGMT

FROM FNF.SAP_FNF.DW_COPA_D
WHERE PST_DT >= '2024-01-01' AND PST_DT <= '2025-11-30'
  AND TO_CHAR(PST_DT, 'MM') <= '11'
GROUP BY TO_CHAR(PST_DT, 'YYYY')
```

```
□ 매출액 = Σ(VAT_EXC_ACT_SALE_AMT)
□ 출고매출 = 매출액 − 점수수료
□ 매출총이익 = 출고매출 − 매출원가
□ 영업이익 = 매출총이익 − 판관비
□ 영업이익률 = 영업이익 ÷ 매출액
□ 브랜드별 합계 = 전체 매출액 (조정 포함)
```

### 9.2 재무상태표 검증
```
□ 자산 = 부채 + 자본
□ 부채비율 = 부채 ÷ 자본
□ 순차입금 = 차입금 − 현금
□ NWC = 재고 + 매출채권 − 매입채무
```

### 9.3 비율 검증
```
□ 수익성 비율은 연환산 기준
□ 안정성 비율은 기말 잔액 기준
□ 활동성 비율은 연환산 매출 ÷ 기말 잔액
```
