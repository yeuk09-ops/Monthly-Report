# FNF 경영실적 대시보드 v2

월별 기준으로 재무 데이터를 관리하고 조회할 수 있는 F&F 경영실적 대시보드입니다.

**Production URL**: https://fnf-dashboardv2.vercel.app

## 주요 기능

- **월별 데이터 선택**: 헤더의 드롭다운으로 조회 기준월 선택
- **경영요약**: KPI, 수익성/안정성/활동성 지표, AI 인사이트
- **재무상태표**: 자산/부채/자본, 운전자본, 여신검증
- **손익계산서**: 매출/비용 구조, 채널별/브랜드별/수출지역별 매출 분석, 브랜드 믹스 차트

## 데이터 관리

### 새 월 데이터 추가

1. `src/data/` 폴더에 `YYYY-MM.json` 파일 생성
2. `src/data/index.json`의 `availableMonths` 배열에 새 월 추가
3. 필요시 `defaultMonth` 업데이트

### JSON 데이터 구조

각 월별 JSON 파일은 다음 구조를 따릅니다:

```json
{
  "meta": { "year": 2025, "month": 12, ... },
  "financialData": { ... },
  "incomeStatement": { ... },
  "channelSales": [ ... ],
  "exportSales": [ ... ],
  "brandSales": [ ... ],
  "balanceSheet": { ... },
  "workingCapital": { ... },
  "creditVerification": [ ... ],
  "aiInsights": { ... }
}
```
